Projeto Mente Em Foco – Saúde Mental e Ansiedade
Site educativo voltado a estudantes, com práticas e informações sobre saúde mental e controle da ansiedade.
Baseado no projeto original “Goblins Lanceiros”, agora modernizado com Next.js 15 e React 18.

Como rodar

cd projeto-2-borboleta-next-app

npm install
npm run dev
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

1. Visão Geral
O Mente em Foco é um recurso online dedicado à promoção da saúde mental, bem-estar emocional e conscientização.
O projeto tem como objetivo fornecer informações acessíveis, ferramentas práticas e um canal de apoio para usuários que buscam melhorar sua qualidade de vida mental.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

2. Recursos Principais
O site oferece as seguintes funcionalidades centrais:

Conteúdo Educacional: Artigos e guias informativos sobre temas como ansiedade, depressão, mindfulness e estratégias de autocuidado.
Ferramentas Práticas: Recursos interativos, como exercícios de respiração guiada e diários de humor para o acompanhamento do estado emocional.
Seção de Apoio: Informações e diretrizes sobre como buscar ajuda profissional, incluindo links para serviços de emergência e linhas de suporte.

---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------

3. Público-Alvo
Indivíduos interessados em saúde mental e autodesenvolvimento, que necessitam de recursos confiáveis e ferramentas para gerenciar seu bem-estar emocional no cotidiano.
