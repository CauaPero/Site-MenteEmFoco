# Projeto Borboleta – Saúde Mental e Ansiedade

Site educativo voltado a estudantes, com práticas e informações sobre saúde mental e controle da ansiedade.  
Baseado no projeto original “Goblins Lanceiros”, agora modernizado com **Next.js 15** e **React 18**.

Como rodar

```bash
cd projeto-2-borboleta-next-app

npm install
npm run dev
